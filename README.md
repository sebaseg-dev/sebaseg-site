# sebaseg-site

## Purpose

For years, I’ve maintained a personal website to showcase my resume and portfolio. My journey into hobby programming began almost 15 years ago, during which I encountered various hosting challenges related to personal information. This is why I decided to keep my previous repository secure. However, I’ve decided to take a fresh start with this project and make it publicly available.

I’m creating a new repository to implement my own custom theme on an Astro base. This presents an excellent opportunity for a fresh start.

## Tech Stack

This website doesn’t require extensive engineering. I prioritize a user-friendly, enjoyable, and maintainable coding experience. Since it’s a static site, I’ll start with a simple Astro.js base.

The website will be continuously deployed on my own Coolify instance.
